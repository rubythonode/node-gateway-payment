# GATEWAY - Testes

Os testes são efetuados para a aplicação rodando em *config.js*.
Resultados dependem dos recursos da Pagar.me


## Instalação do ambiente

- Node.js versão 6.x
- Mocha
- Supertest
- Superagent

```bash
npm instal -g mocha
cd funcionais
npm install
```

## Como usar

- Acessar a pasta SRC do projeto
- Rodar o Script

```bash
npm test
```

ou

- Acessar a pasta dos testes do projeto
- Rodar o Script

```bash
npm start
```
